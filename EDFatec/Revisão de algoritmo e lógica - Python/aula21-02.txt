matéria aula:
Funções recursivas
função que se chama a si propria
dado que contén um subdado que é igualzinho
sempre teho um caso simples onde sei a resposta de n!
	
	1 caso n seja 0 ou 1
	n! = n*(n-1)!
Ex.: 
	def fat(n):
	  if n == 0 or n == 1: return 1
	  return n * fat(n-1)
	print(fat(4)) #repetição feita usando dado
	
	def pot(x, n): #2**3 = 8 = pot(2, 3)
	  if n == 0: return 1
	  return x * pot(x, n-1)
	print (pot(2, 3))
